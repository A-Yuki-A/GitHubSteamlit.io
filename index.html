アプリ 公開手順ガイド

GitHub と Streamlit Community Cloud を使って、PythonアプリをWeb公開する手順をまとめたページです。

0. 全体の流れ

  1. GitHubにリポジトリ（プロジェクト用フォルダ）を作成する
  2. GitHub上でアプリ用ファイルを作成する
  3. Streamlit Community Cloud でアプリをデプロイ（公開）する

1. 事前準備

  • GitHubアカウントを作成する（https://github.com/）
  • Streamlitアカウントを作成する（https://streamlit.io/ → GitHubでログイン）
  • GitHub や Streamlit のアカウント登録には、個人のメールアドレスを使用してください。
    例：Gmail、Yahooメールなど。
    学校のメールアドレスや学校アカウントは使用しないでください。

2. GitHubでリポジトリを作る

2-1. リポジトリとは？

リポジトリ（repository）とは、GitHub上に作る「プロジェクト用フォルダ」です。
アプリや作品ごとにリポジトリを分けて管理します。

2-2. リポジトリ作成手順

  1. GitHubにログインし、右上の「＋」→ New repository をクリックする
  2. Repository name に、例：my-streamlit-app を入力する
  3. Description アプリの説明を入力　例：ドラクエ風ガチャコード
  4. Public を選択する。（GituHubは公開が基本）
  5. Add a README file にチェックを入れる
  6. Create repository をクリックする

3. GitHub上でファイルを作成する（Create new file）

3-1. app.py を作成

  1. 作成したリポジトリのページを開く
  2. Add file → Create new file を選択する
  3. File name に app.py と入力する
  4. 下のテキストエリアにコードを貼り付ける
  5. Commit new file を押して保存する

3-2. requirements.txt を作成

  1. Add file → Create new file を選択する
  2. File name に requirements.txt と入力する
  3. 次の1行を入力する：
streamlit
  1. Commit new file を押して保存する

4. アプリ用ファイルと拡張子の説明

4-1. 拡張子とは？

ファイル名の最後の .py や .txt を拡張子と呼びます。 拡張子は「そのファイルが何の種類か」を表します。

  • .py（Pythonファイル）
    Pythonのプログラムを書くファイルです。 Streamlitアプリ本体は app.py に書きます。
  • .txt（テキストファイル）
    文字だけを保存するファイルです。 使用するライブラリの一覧を requirements.txt に書きます。

5. Streamlit Community Cloudで公開する

  1. Streamlit Community Cloud にアクセスし、GitHubでログインする
  2. New app をクリックする
  3. Repository に ユーザー名/リポジトリ名 を指定する
  4. Branch は main、Main file path は app.py を指定する
  5. Deploy を押すと公開URLが発行される
