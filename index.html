Streamlitアプリ 公開手順ガイド

GitHub と Streamlit Community Cloud を使って、PythonアプリをWeb公開する手順をまとめたページです。

0. 全体の流れ

  1. GitHubにリポジトリ（プロジェクト用フォルダ）を作成する
  2. リポジトリの中に app.py と requirements.txt を作成する
  3. Streamlit Community Cloud でアプリをデプロイ（公開）する

1. 事前準備

  • GitHubアカウントを作成する（https://github.com/）
  • Streamlitアカウントを作成する（https://streamlit.io/ → GitHubでログイン）
  • GitHub や Streamlit のアカウント登録には、必ず個人のメールアドレスを使うこと。
    例：Gmail、Yahooメールなど。
    学校のメールアドレスや学校アカウント（例：〇〇高校ドメイン、oughアカウント）は使用不可。
    学校アカウントは外部サービスとの連携制限があり、 GitHub・Streamlit にログインできない場合があります。

2. GitHubでリポジトリを作る

2-1. リポジトリとは？

リポジトリ（repository）とは、GitHub上に作る「プロジェクト用フォルダ」です。
アプリや作品ごとにリポジトリを分け、ファイルをまとめて保存します。

2-2. リポジトリ作成手順

  1. GitHubにログインし、右上の「＋」→ New repository をクリック
  2. Repository name に例：my-streamlit-app を入力する
  3. Public を選択する
  4. 必要なら「Add a README file」にチェック
  5. Create repository をクリックする

3. アプリ用ファイルと拡張子の説明

3-1. 拡張子とは？

ファイル名の最後の .py や .txt を拡張子と呼びます。 「どんな種類のファイルか」を表す印です。

  • .py（Pythonファイル）
    Pythonプログラムを書くファイル。 Streamlitアプリ本体は app.py で作成します。
  • .txt（テキストファイル）
    文字だけを保存するファイル。 Streamlitで使うライブラリ一覧を requirements.txt に書きます。

3-2. app.py のサンプルコード

import streamlit as st

st.title("勇者のステータス表示アプリ")

namae = st.text_input("勇者の名前を入力してください")
level = st.slider("レベル", 1, 99, 10)
hp = st.number_input("HP", min_value=1, max_value=999, value=100)

if st.button("ステータスを表示"):
    st.write("=== 勇者のステータス ===")
    st.write("名前：", namae)
    st.write("レベル：", level)
    st.write("HP：", hp)

3-3. requirements.txt の内容

streamlit

3-4. 生成AIを使ってコードや説明文も作れる

Streamlitアプリのコードや説明文は、生成AI を使うと簡単に作成できます。 代表的なAIサービスは次の3つです：

  • ◆ ChatGPT（OpenAI）
    コード作成、バグの原因説明、文章生成、画像生成が得意。
    「こういうアプリを作りたい」と言えば、サンプルコードを自動生成。
    公式サイト： https://chat.openai.com
  • ◆ Gemini（Google）
    分かりやすい説明が得意で、文章の要約・改善にも強い。
    Googleアカウントと連携しやすく、調べ学習にも向いています。
    公式サイト： https://gemini.google.com
  • ◆ Claude（Anthropic）
    長文の理解力が高く、正確なコード説明や添削が得意。
    大量の文章を扱う作業（レポート、仕様書、コード解析）にも強いAIです。
    公式サイト： https://claude.ai

生成AIは アイデア作成・コードのたたき台作成・エラーの原因調査 に非常に便利です。 ただし、AIの出力が間違っていることもあるので、必ず実行して確認しましょう。

4. GitHub上でファイルを作成する（Create new file）

4-1. app.py を作成

  1. 作成したリポジトリのページを開く
  2. 右上の Add file → Create new file を選択
  3. File name に app.py と入力
  4. 下のテキストエリアにコードを貼り付ける
  5. 下部の Commit new file を押して保存

4-2. requirements.txt を作成

  1. Add file → Create new file を選択
  2. File name に requirements.txt と入力
  3. 次の1行を入力する：
streamlit
  1. Commit new file を押して保存

5. Streamlit Community Cloudで公開する

  1. Streamlit Community Cloud にアクセスし、GitHubでログインする
  2. New app をクリックする
  3. Repository に ユーザー名/リポジトリ名（例：testuser/my-streamlit-app）を選択
  4. Branch は main、Main file path は app.py を指定
  5. Deploy を押すと公開URL（https://～.streamlit.app）が発行される
