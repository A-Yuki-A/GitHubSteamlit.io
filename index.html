Streamlitアプリ 公開手順ガイド

GitHub と Streamlit Community Cloud を使って、PythonアプリをWeb公開する手順をまとめました。

0. 全体の流れ

  1. GitHubにリポジトリを作成する
  2. Streamlitアプリのファイル（app.py と requirements.txt）を作成してGitHubにアップする
  3. Streamlit Community Cloud でアプリをデプロイする
  4. （任意）このページのような説明用サイトを GitHub Pages で公開する

1. 事前準備

  • GitHubアカウントの作成（https://github.com/）
  • Streamlitアカウントの作成 （https://streamlit.io/ → GitHubでログイン）
  • PCにPythonをインストール（ローカルでテストしたい場合）

2. GitHubでリポジトリを作る

  1. GitHubにログインし、右上の「＋」→ New repository をクリック
  2. Repository name に例：my-streamlit-app を入力
  3. Public を選んで、必要なら「Add a README file」にチェック
  4. Create repository をクリック

3. Streamlitアプリのファイルを作る

3-1. app.py の例

import streamlit as st

st.title("勇者のステータス表示アプリ")

namae = st.text_input("勇者の名前を入力してください")
level = st.slider("レベル", 1, 99, 10)
hp = st.number_input("HP", min_value=1, max_value=999, value=100)

if st.button("ステータスを表示"):
    st.write("=== 勇者のステータス ===")
    st.write("名前：", namae)
    st.write("レベル：", level)
    st.write("HP：", hp)

streamlit

他に pandas や numpy などを使う場合は、1行ずつ追加して書きます。

4. ファイルをGitHubにアップロードする

ブラウザからアップロードする場合

  1. リポジトリのページを開く
  2. Add file → Upload files をクリック
  3. app.py と requirements.txt をドラッグ＆ドロップ
  4. 下のメッセージ欄に簡単な説明を書いて、Commit changes をクリック

5. Streamlit Community Cloudで公開する

  1. Streamlit Community Cloud にアクセスし、GitHubでログイン
  2. New app ボタンをクリック
  3. Repository に先ほど作成したリポジトリ（例：ユーザー名/my-streamlit-app）を選択
  4. Branch は通常 main、Main file path は app.py を指定
  5. Deploy を押すとビルドが始まり、完了すると https://〜.streamlit.app のURLが発行される

このURLを生徒に配布すると、ブラウザからアプリを使えるようになります。

6. この説明ページをGitHub Pagesで公開する

  1. この index.html と style.css をリポジトリに置く
  2. GitHubのリポジトリで Settings → Pages を開く
  3. Source で Branch を main、フォルダを /root などに設定し、保存
  4. しばらくすると、GitHub Pages のURL（例：https://ユーザー名.github.io/リポジトリ名/）が発行される

このページ自体が、GitHub Pages で公開された「アプリの公開手順ガイド」になります。
