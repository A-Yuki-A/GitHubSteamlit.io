Streamlitアプリ 公開手順ガイド

GitHub と Streamlit Community Cloud を使って、PythonアプリをWeb公開する手順をまとめました。

0. 全体の流れ

  1. GitHubにリポジトリ（フォルダ）を作成する
  2. Streamlitアプリのファイル（app.py と requirements.txt）を作成してGitHubに保存する
  3. Streamlit Community Cloud でアプリをデプロイ（公開）する
  4. （任意）この説明ページを GitHub Pages で公開する

1. 事前準備

  • GitHubアカウントの作成（https://github.com/）
  • Streamlitアカウントの作成（https://streamlit.io/ → GitHubでログイン）
  • アカウント登録には、個人のメールアドレス（Gmail など）を使用すること。
    学校のメールアドレス（@◯◯.ed.jp など）は、外部サービスとの連携に制限があり、 GitHub・Streamlitでログインできないことがあるため使用できない。

2. GitHubでリポジトリを作る（リポジトリとは？）

● リポジトリとは？

GitHubでいう「リポジトリ」は、プロジェクトをまとめて保存するフォルダのようなものです。
Streamlitアプリを公開するためには、アプリを構成するファイル一式を GitHub上のリポジトリにまとめて保存します。

● リポジトリ作成手順

  1. GitHubにログインし、右上の「＋」→ New repository をクリック
  2. Repository name に例：my-streamlit-app を入力
  3. Public を選択（公開アプリの場合）
  4. 必要なら「Add a README file」にチェック
  5. Create repository をクリック

3. Streamlitアプリのファイルを作る

● ファイル形式について

  • .py …… Pythonプログラムのファイル
    Streamlitアプリ本体（プログラム）はこの形式で作成します。
  • .txt …… テキストファイル
    requirements.txt のように、アプリで使うライブラリ一覧を記述するために使用します。

● 3-1. app.py（アプリ本体）の例

import streamlit as st

st.title("勇者のステータス表示アプリ")

namae = st.text_input("勇者の名前を入力してください")
level = st.slider("レベル", 1, 99, 10)
hp = st.number_input("HP", min_value=1, max_value=999, value=100)

if st.button("ステータスを表示"):
    st.write("=== 勇者のステータス ===")
    st.write("名前：", namae)
    st.write("レベル：", level)
    st.write("HP：", hp)

● 3-2. requirements.txt の例

streamlit

他のライブラリ（例：pandas, numpy）を使う場合は1行ずつ追加します。

4. GitHub上でファイルを作成する（Create new file）

● 4-1. app.py を作成

  1. リポジトリのページを開く
  2. 右上の Add file → Create new file をクリック
  3. 「File name」に app.py と入力
  4. 下の大きな入力欄に、先ほどの app.py のコードを貼り付ける
  5. 下へスクロールし、「Commit new file」をクリックして保存

● 4-2. requirements.txt を作成

  1. 再度 Add file → Create new file をクリック
  2. 「File name」に requirements.txt と入力
  3. 内容欄に次の1行を入力
streamlit
  1. 「Commit new file」をクリックして保存

5. Streamlit Community Cloudで公開する

  1. Streamlit Community Cloud にアクセスし、GitHubでログイン
  2. New app ボタンをクリック
  3. Repository に先ほどのリポジトリ（例：ユーザー名/my-streamlit-app）を指定
  4. Branch は main、Main file path は app.py を選択
  5. Deploy を押すとビルドが始まり、公開用URL（https://～.streamlit.app）が発行される

6. この説明ページをGitHub Pagesで公開する

  1. index.html と style.css をリポジトリに追加
  2. GitHubの Settings → Pages を開く
  3. Source を main、フォルダを /root に設定して保存
  4. 表示されたURL（例：https://ユーザー名.github.io/リポジトリ名/）にアクセス
