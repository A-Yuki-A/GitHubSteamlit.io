Streamlitアプリ 公開手順ガイド

GitHub と Streamlit Community Cloud を使って、PythonアプリをWeb公開する手順をまとめたページです。

0. 全体の流れ

  1. GitHubにリポジトリ（プロジェクト用フォルダ）を作成する
  2. リポジトリの中に app.py と requirements.txt を作成する
  3. Streamlit Community Cloud でアプリをデプロイ（公開）する

1. 事前準備

  • GitHubアカウントを作成する（https://github.com/）
  • Streamlitアカウントを作成する（https://streamlit.io/ → GitHubでログイン）
  • GitHub や Streamlit のアカウント登録には、必ず個人のメールアドレスを使うこと。
    例：Gmail、Yahooメールなど。
    学校のメールアドレスや学校のアカウント（例：〇〇高校のアカウント、oughアカウントなど）は使用できません。
    学校アカウントは外部サービスとの連携が制限されており、 GitHub や Streamlit にログインできない場合があるためです。

2. GitHubでリポジトリを作る

2-1. リポジトリとは？

リポジトリ（repository）とは、GitHub上に作る「プロジェクト用のフォルダ」のようなものです。
1つのアプリや作品ごとに1つのリポジトリを作り、その中にファイルをまとめて保存します。
このページで作る app.py と requirements.txt も、同じリポジトリの中に置きます。

2-2. リポジトリ作成手順

  1. GitHubにログインし、右上の「＋」→ New repository をクリック
  2. Repository name に例：my-streamlit-app を入力する
  3. Public を選択する（公開したい場合）
  4. 必要なら「Add a README file」にチェックを入れる
  5. Create repository をクリックする

3. アプリ用ファイルと拡張子の説明

3-1. 拡張子とは？

ファイル名の最後についている .py や .txt を 拡張子（かくちょうし） と呼びます。
拡張子によって「このファイルは何の種類か（プログラムか、文章か）」を区別します。

  • .py（Pythonファイル） Pythonというプログラミング言語で書かれたプログラムファイル。 Streamlitアプリの本体は app.py という .py ファイルとして作ります。
  • .txt（テキストファイル） 文字だけの内容を保存するファイル。 Streamlitでは、使用するライブラリ一覧を requirements.txt として記述します。

3-2. app.py のサンプルコード

import streamlit as st

st.title("勇者のステータス表示アプリ")

namae = st.text_input("勇者の名前を入力してください")
level = st.slider("レベル", 1, 99, 10)
hp = st.number_input("HP", min_value=1, max_value=999, value=100)

if st.button("ステータスを表示"):
    st.write("=== 勇者のステータス ===")
    st.write("名前：", namae)
    st.write("レベル：", level)
    st.write("HP：", hp)

3-4. 生成AIを使ってコードや説明文も作れる

プログラムのコード、説明文、デザインの提案などは、生成AI を使うことで簡単に作ることができます。 特に次の2つがよく使われています：

  • ◆ ChatGPT（OpenAI）
    コード生成、エラーメッセージの解説、文章作成、画像生成などが得意。
    「このようなStreamlitアプリを作りたい」と言えば、コード例を作ってくれます。
  • ◆ Gemini（Google）
    文章生成や説明が得意で、コード例やWebページの修正案も作成可能。
    Googleアカウントと連携しやすいため、資料作りにも向いています。

生成AIは、アイデア作り・コードのたたき台作り・エラーの原因調査 などに非常に役立ちます。 ただし、AIの回答が間違っていることもあるため、必ず実行して確認しましょう。

4. GitHub上でファイルを作成する（Create new file）

4-1. app.py を作成

  1. 作成したリポジトリのページを開く
  2. ファイル一覧の右上にある Add file → Create new file をクリック
  3. 「File name」に app.py と入力する
  4. テキストエリアに app.py の内容を貼り付ける
  5. 「Commit new file」をクリックし保存する

4-2. requirements.txt を作成

  1. Add file → Create new file をクリック
  2. 「File name」に requirements.txt と入力する
  3. 内容欄に次の1行を入力する
streamlit
  1. 「Commit new file」をクリックして保存する

5. Streamlit Community Cloudで公開する

  1. Streamlit Community Cloud にアクセスし、GitHubでログインする
  2. New app をクリックする
  3. Repository に作成したリポジトリ（例：ユーザー名/my-streamlit-app）を選択
  4. Branch は main、Main file path は app.py を指定
  5. Deploy を押すとビルドが始まり、公開URL（https://〜.streamlit.app）が発行される
