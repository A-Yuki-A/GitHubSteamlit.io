アプリ 公開手順ガイド

GitHub と Streamlit Community Cloud を使って、Python（Streamlit）アプリをWeb公開する手順をまとめたページです。

0. 全体の流れ

  1. GitHubにリポジトリ（プロジェクト用フォルダ）を作成する
  2. GitHub上でアプリ用ファイル（app.py, requirements.txt）を作成する
  3. 生成AIを使ってコードを作成・改善する
  4. Streamlit Community Cloud でアプリをデプロイ（公開）する

1. 事前準備

  • GitHubアカウントを作成する（https://github.com/）
  • Streamlitアカウントを作成する（https://streamlit.io/ → GitHubでログイン）
  • GitHub や Streamlit のアカウント登録には、個人のメールアドレスを使用してください。
    例：Gmail、Yahooメールなど。
    学校のメールアドレスや学校アカウントは使用しないでください。

2. GitHubでリポジトリを作る

2-1. リポジトリとは？

リポジトリ（repository）とは、GitHub上に作る「プロジェクト用フォルダ」です。
アプリや作品ごとにリポジトリを分けて管理します。

2-2. リポジトリ作成手順

  1. GitHubにログインし、右上の「＋」→ New repository をクリックする
  2. Repository name にアプリ名、例：dorakue-gatya を入力する
  3. Description にアプリの説明を入力する　例：ドラクエ風ガチャアプリ
  4. Public を選択する（公開して困る情報は置かない）
  5. Add a README file にチェックを入れる
  6. Create repository をクリックする

3. GitHub上でファイルを作成する（Create new file）

3-1. app.py を作成

  1. 作成したリポジトリのページを開く
  2. Add file → Create new file を選択する
  3. File name に app.py と入力する
  4. Enter file contents here のテキストエリアに作成したコードを貼り付ける
  5. Commit changes… を押して保存する

3-2. requirements.txt を作成

  2. app.py を開く

4. 生成AIを使ってコードを作成する

4-1. 使えるサービス

  • ChatGPT（OpenAI）： https://chat.openai.com/
  • Claude（Anthropic）： https://claude.ai/

4-2. 基本の流れ

  1. 「作りたいアプリの内容」を文章で整理する（目的・入力・出力）
  2. 生成AIにプロンプト（指示文）を送る
  3. 出てきたコードを app.py に貼り付ける
  4. 動かして確認し、エラーや改善点をAIに相談する

4-3. プロンプトの書き方（例）

まずは次の形が分かりやすいです。

〇〇のアプリをPythonで作りたいです。
条件：
- Streamlit を使う
- ファイル名は app.py
- 入力：＿＿＿＿
- 出力：＿＿＿＿
- 初心者向けにコメントも入れて
コードを全文でください。

例：ガチャアプリの場合

ドラクエ風ガチャのアプリをPythonで作りたいです。
条件：
- Streamlit を使う
- ファイル名は app.py
- ボタンを押すと「S/A/B/C」のどれかを確率で表示する（例：S 5%, A 15%, B 30%, C 50%）
- 履歴も画面に残るようにする
- 初心者向けにコメントも入れて
コードを全文でください。

4-4. うまくいかないときの追加プロンプト（例）

エラーが出ました。下のエラー文を読んで原因と直し方を教えてください。
（ここにエラー文を貼る）

生成AIは便利ですが、間違ったコードになることもあります。必ずコードを読み、実行して確認しましょう。

5. アプリ用ファイルと拡張子の説明

5-1. 拡張子とは？

ファイル名の最後の .py や .txt を拡張子と呼びます。 拡張子は「そのファイルが何の種類か」を表します。

  • .py（Pythonファイル）
    Pythonのプログラムを書くファイルです。 Streamlitアプリ本体は app.py に書きます。
  • .txt（テキストファイル）
    文字だけを保存するファイルです。 使用するライブラリの一覧を requirements.txt に書きます。

6. Streamlit Community Cloudで公開する

  1. Streamlit Community Cloud にアクセスし、GitHubでログインする
  2. Create app をクリックする
  3. Deploy a public app from GitGub をクリックする
  4. Repository に GitHubで作成したファイル を選択する
  5. Branch は main、Main file path は app.py を指定する
  6. Deploy を押すと公開URL（https://～.streamlit.app）が発行される
